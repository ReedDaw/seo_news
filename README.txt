INTRODUCTION
------------

The URL Opener Script opens up a list of URLs in a new tab of an existing Chrome window.


REQUIREMENTS
------------

This module requires the following external modules:

 * webbrowser (documentation: https://docs.python.org/2/library/webbrowser.html)

-------------
